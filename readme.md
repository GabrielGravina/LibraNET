Entrar no diretório backend e: <br/>
criar ambiente virtual com: <br/>
```
python -m venv venv
```
Ativar ambiente virtual <br/>
Instalar requirements.txt
Executar: <br/>
```
set FLASK_APP=app.py
set FLASK_ENV=development
```
```
flask run --reload
```
<br/>
Entrar no diretório frontend e:<br/>

```
npm install
npm run dev
```
